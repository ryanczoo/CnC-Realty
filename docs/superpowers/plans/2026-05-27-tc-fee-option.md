# TC Fee Option Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an optional $350 CnC TC Service fee to the transaction file — toggled by agents during transaction creation, stored on the record, shown in the commission breakdown, and reflected in the net-to-agent calculation.

**Architecture:** Single boolean field `tcFeeEnabled` added to `TransactionFile`. The $350 amount is a constant in code — not stored separately — so the fee can be adjusted brokerage-wide without a migration. The toggle appears in the wizard Commission step, flows through the API, updates the type definition, and renders in the CommissionTab on the transaction detail page.

**Tech Stack:** Prisma ORM, Next.js 14 App Router, TypeScript, Tailwind CSS
